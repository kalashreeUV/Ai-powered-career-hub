About AI Powered Career Hub

The AI Powered Career Hub is a professional platform designed to help students and job seekers manage and track their skills, education, and career goals. This project demonstrates a complete full-stack application using Python, Flask, and SQLite, with a user-friendly interface.

Features:

  • Register and store user information including skills, education, and goals.
  • View all registered users in a dynamic table with clickable rows.
  • Modal popup to view, edit, or delete user details seamlessly.
  • Search and filter users based on name, skills, or goals.
  • Future-ready structure to integrate AI-powered resume analysis and career guidance.

Tech Stack:

  • Backend: Python, Flask, SQLite
  • Frontend: HTML, CSS, JavaScript
  • API: RESTful endpoints for user management
  • Additional: Modal UI, search/filter, CRUD operations

This platform can be extended further to include AI-based career recommendations, resume scanning, and interview preparation guidance, making it a professional-grade project for your portfolio.
